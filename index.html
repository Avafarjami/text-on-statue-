// تنظیم ابعاد برچسب (مثلاً ۵۰ سانتی‌متر عرض) // این اعداد را بر اساس بزرگی شیء واقعی‌ات تغییر بده const label = new THREE.Mesh(new THREE.PlaneGeometry(0.5, 0.125), material); // قرار دادن دقیق متن در محل کلیک label.position.copy(point); // مهم: چسباندن متن به سطح با استفاده از بردار Normal // متن را طوری می‌چرخانیم که رو به جهت بیرونِ وجه باشد const lookAtPoint = new THREE.Vector3().addVectors(point, normal); label.lookAt(lookAtPoint); // یک فاصله بسیار کم (Offset) برای جلوگیری از تداخل با سطح مجسمه (چشمک زدن تصویر) label.position.add(normal.clone().multiplyScalar(0.01)); // اضافه کردن به Mesh مورد نظر، نه مستقیم به Scene، تا با حرکت مدل جابه‌جا شود targetMesh.add(label); }
